Macro Overlay for Windows
=========================

Run MacroOverlay.exe to start the overlay. No Java installation is required.

While the app is running, its blue lightning icon appears in the Windows
notification area. Double-click the icon or choose "Show Main Controls" to
restore the main ImGui window. Choose "Exit Macro Overlay" to close the app
from anywhere, even when all overlay panels are hidden.

The first launch may display a Windows SmartScreen warning because this build
is not code-signed. Use "More info" and then "Run anyway" if you trust the
person who shared this package with you.

Settings and macro profiles are stored per Windows user in LocalAppData.
